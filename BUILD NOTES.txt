BUILD NOTES

GAME OVERVIEW

The current stage is just a sandbox of the key feature and a skeleton of what the game will look like. It is a rhythm game with 4 rails where notes drop and need to be hit while being on beat with different tracks. A percentage based score will keep track of how accurate you are, with the aim of being as accurate as possible.

CURRENT FEATURES:

The features currently available to the player are the following: the ability to use the keyboard to hit notes, with visual feedback when pressing the button associated with a rail. If the note is missed, then it will disappear when hit outside the zone. If a note is hit within the hit zone, then the notes gets destroyed on impact. The notes are currently randomized on the tracks.

CONTROLS:

D for the first rail.
F for the second rail.
J for the third rail.
K for the fourth rail.
Q to quit the game.
